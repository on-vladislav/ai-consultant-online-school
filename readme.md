# Сайт с ИИ-консультантом для онлайн-школы (Voiceflow + OpenAI)

## Цель
Создать сайт онлайн-школы в Creatium и встроить ИИ-консультанта (виджет Voiceflow),
который отвечает на FAQ, помогает выбрать курс и собирает заявки.

## Стек
- Creatium (сайт)
- Voiceflow (виджет/логика бота)
- OpenAI API (LLM)

## Что будет в репозитории
- /docs — логика ассистента, база знаний, тестирование, отчет
- /prompts — системный промпт и промпты диалогов
- /screenshots — скриншоты Voiceflow и сайта
- /commercial_offer — коммерческое предложение
