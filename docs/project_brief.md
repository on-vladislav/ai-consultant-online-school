# Project brief

## Проблема
Пользователи онлайн-школы нуждаются в быстрой поддержке при выборе курса, уточнении расписания, стоимости и записи.

## Пользователи
- Потенциальные студенты
- Текущие студенты
- Администраторы (получают заявки)

## Основные функции ассистента
- FAQ (цена, расписание, формат, длительность, старт)
- Рекомендации курса по запросу пользователя
- Сбор заявки (имя, контакт, интересующий курс/уровень)
- Возможность подключить базу знаний

## Срок
2 недели
